## 📌 과정 개요

본 과정은 SW 개발자를 대상으로 AI를 활용한 코드 리팩토링 및 코드 리뷰 역량 강화 방법을 학습한다.

---

## 🎯 학습 목표

- 클린코드, 리팩토링, TDD(테스트 주도 개발)의 개념을 실습하고, 해당 기술들을 실제 프로젝트에 적용할 수 있다.
- 생성형 AI 기반 코드 리팩토링, 디버깅, 테스트 코드를 작성할 수 있다.
- AI를 활용한 코드 리뷰 및 품질을 개선할 수 있다.
- 개발 프로세스 내 AI 활용 지점 파악과 실무 적용 능력을 배양할 수 있다.
