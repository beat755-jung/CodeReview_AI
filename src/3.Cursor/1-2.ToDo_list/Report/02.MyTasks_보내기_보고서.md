## 2. 핵심 기능 한눈에 보기

- 할 일 **추가 / 삭제 / 완료** 및 목록 표시  
- **카테고리:** 업무(`work`), 개인(`personal`), 공부(`study`) — 추가 시 선택, 행마다 색 태그  
- **필터:** 전체·카테고리별 — 상단 버튼, 선택 상태 강조 및 `localStorage`에 유지  
- **생성 시각:** 상대 표현(예: "2시간 전")  
- **정렬:** 미완료 상단, 완료 하단(같은 그룹 내 생성 순)  
- **이전 버전 데이터:** `myTasks:v1` → 최초 로드 시 `myTasks:v2`로 이전 가능

---

## 3. localStorage 키 요약

| 키 | 용도 |
|----|------|
| `myTasks:v2` | 할 일 배열(JSON) |
| `myTasks:filter:v1` | 현재 필터 값(JSON 문자열) |

---

## 4. 인수 시 확인 권장 사항

1. 브라우저에서 `index.html`을 연 뒤 할 일 추가·필터 전환·새로고침으로 데이터 유지 여부를 확인한다.  
2. 예전에 `myTasks:v1`만 쓰던 환경이라면, 한 번 로드 후 v2에 마이그레이션되었는지 개발자 도구 Application 탭에서 확인할 수 있다.  
3. 배포 시에는 정적 호스팅 또는 웹 서버에 세 파일을 함께 올리고, `index.html`이 `style.css`·`script.js`와 상대 경로로 맞는지 확인한다.
