**User**

다크 모드와 추가 기능들을 구현해줘.
요구사항:
1. 다크 모드:
   - 우측 상단에 다크/라이트 모드 토글 스위치
   - 다크 모드 색상: 배경(#1A1A1A), 카드(#2D2D2D), 텍스트(#E0E0E0)
   - 선택한 테마 localStorage에 저장
   - 부드러운 전환 애니메이션
2. 추가 기능:
   - "완료된 항목 모두 삭제" 버튼 (확인 다이얼로그 포함)
   - 할 일 검색 기능 (실시간 필터링)
   - 남은 할 일 개수 배지 표시
   - 빈 상태 메시지 ("할 일이 없습니다. 추가해보세요!")
3. 키보드 단축키:
   - Alt+N: 새 할 일 입력창 포커스
   - Alt+1,2,3,4: 카테고리 필터 전환
   - Alt+D: 다크 모드 토글
4. 반응형 디자인:
   - 모바일 (max-width: 480px) 최적화
   - 터치 친화적인 버튼 크기
모든 인터랙션에 적절한 피드백을 제공해줘.

**Cursor**

프로젝트 구조와 기존 코드를 확인 중입니다.



구조를 파악했습니다. 다크 모드, 검색, 배지, 완료 항목 일괄 삭제, 키보드 단축키, 반응형·피드백(토스트·`<dialog>`)을 구현합니다.



`script.js`에 테마·검색·배지·다이얼로그·단축키·토스트 로직을 통합합니다.



Alt+N의 토스트는 포커스만으로 충분해 제거하고, `render()`마다 배지 펄스가 나가지 않도록 카운트가 바뀔 때만 펄스합니다. 사용하지 않는 `isTypingTarget` 함수를 제거합니다.



파일에 줄바꿈이 이중으로 들어가 있습니다. 원래 스타일로 `script.js`를 정리해 다시 씁니다.



`index.html`을 읽기 좋은 한 줄 간격으로 정리합니다.



`style.css`의 연속 빈 줄을 정리하는 중입니다.



구현을 모두 반영했습니다. 요약만 정리합니다.

## 1. 다크 모드
- 헤더 우측에 **토글 스위치**(체크박스 + 라벨 트랙, `title`에 Alt+D 안내).
- 다크 시 CSS 변수: 배경 `#1A1A1A`, 카드 `#2D2D2D`, 본문 `#E0E0E0` (`html[data-theme="dark"]`).
- 선택값은 **`localStorage` 키 `myTasks:theme:v1`** (`light` / `dark`).
- `head` 안 **인라인 스크립트**로 초기 로드 시 깜빡임을 줄입니다.
- `background`, `color`, `border-color` 등에 **`--theme-transition`(0.28s)** 적용.

## 2. 추가 기능
- **완료된 항목 모두 삭제**: `<dialog>` + `method="dialog"` 폼으로 확인 후 일괄 삭제. 완료 항목이 없으면 버튼 **비활성화**.
- **검색**: `#task-search`에서 입력 시 **즉시** 목록 필터(대소문자 무시, 제목 부분 일치).
- **남은 할 일 배지**: 현재 **카테고리 필터** 기준 미완료 개수 + 필터 이름(예: `남은 할 일 3개 · 업무`). 개수가 바뀔 때만 짧은 **펄스** 애니메이션.
- **빈 목록**: 할 일 0개일 때 **「할 일이 없습니다. 추가해보세요!」** (요구사항 문구).

## 3. 키보드
- **Alt+N**: `#task-input` 포커스.
- **Alt+1~4**: 전체 / 업무 / 개인 / 공부.
- **Alt+D**: 다크 모드 토글(스위치·저장·토스트와 동일 동작).

## 4. 반응형·터치
- `@media (max-width: 480px)`: 여백·타이포·툴바 세로 배치·필터 2열·일괄 삭제·폼 제출 전체 폭 등.
- **`--touch-min: 44px`**: 필터, 폼 컨트롤, 삭제 버튼, 다이얼로그 버튼 등에 최소 높이 반영.

## 5. 피드백
- 하단 **토스트**: 할 일 추가·수정·삭제·완료 일괄 삭제·테마 전환.
- 버튼 **호버/액티브/포커스 표시**, 폼 **`focus-within`**, 삭제 버튼 호버 등 기존 패턴 유지·확장.

수정·추가된 파일: `index.html`, `style.css`, `script.js`. 브라우저에서 `index.html`을 열어 테마·검색·단축키·480px 이하 폭을 한번씩 확인해 보시면 됩니다.
