## 생성된 파일

- **`index.html`** — 제목 "My Tasks", `#task-form`(입력 + 추가), `#task-list` 목록
- **`style.css`** — 카드형 레이아웃, 본문 중앙·`max-width: 600px`, 호버·`transition`
- **`script.js`** — CRUD + `localStorage` 연동

## 데이터 형식

각 항목은 다음 형태로 배열에 넣어 `myTasks:v1` 키로 저장합니다.

- `id` — `crypto.randomUUID()`(없으면 대체 문자열)
- `text` — 할 일 문자열
- `completed` — 완료 여부
- `createdAt` — `new Date().toISOString()`

추가·삭제·체크 변경 시마다 `saveTasks()`로 저장하고, 로드 시 형식이 맞는 항목만 복원합니다.

## Enter 키

`<form>` 안의 텍스트 입력에서 **Enter는 브라우저가 폼 제출**로 처리하므로, `submit` 한 곳에서만 `addTask`를 호출해 **중복 추가 없이** Enter와 버튼 모두 지원합니다.

로컬에서 `index.html`을 브라우저로 열면 바로 사용할 수 있습니다.
