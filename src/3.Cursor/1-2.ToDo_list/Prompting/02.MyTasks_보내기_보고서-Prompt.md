## 구현 요약

### 1. 카테고리
- **업무(work)**, **개인(personal)**, **공부(study)** — 추가 폼의 `<select>`로 선택
- 각 항목에 카테고리 라벨 태그(요청하신 색: `#4A90E2`, `#27AE60`, `#8E44AD`)
- 헤더 아래 **전체 / 업무 / 개인 / 공부** 필터 버튼

### 2. UI
- 필터는 **제목 바로 아래** 카드 안에 배치
- 선택된 필터는 `task-filter__btn--active`로 **강조**(배경·테두리·그림자)
- `<time>` + `formatRelativeTime()`으로 **「방금 전」「N분 전」「N시간 전」** 등 한국어 상대 시간 표시

### 3. 데이터·저장
- 할 일 객체에 **`category`** 필드 (`work` | `personal` | `study`)
- 저장 키를 **`myTasks:v2`**로 올렸고, **`myTasks:v1`**만 있으면 자동으로 읽어 `category: "work"`로 옮긴 뒤 v2에 저장
- 필터는 **`myTasks:filter:v1`**에 `JSON.stringify`로 저장

### 4. 정렬
- `sortTasksForDisplay`: **미완료가 위**, **완료는 아래**; 각 그룹 안에서는 **생성일(createdAt) 오름차순**

관련 코드 위치:

```53:75:c:\DEV\ToDo_list\script.js
  function sortTasksForDisplay(arr) {
    return arr.slice().sort(function (a, b) {
      if (a.completed !== b.completed) return a.completed ? 1 : -1;
      return new Date(a.createdAt) - new Date(b.createdAt);
    });
  }

  function getFilteredTasks() {
    if (activeFilter === "all") return tasks.slice();
    return tasks.filter(function (t) {
      return t.category === activeFilter;
    });
  }
```
